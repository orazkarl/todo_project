# To-Do List

## Установка

1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Настройка PostgreSQL в `settings.py`.

4. Миграция:
    ```bash
    python manage.py migrate
    ```

5. Создание суперюзера:
    ```bash
    python manage.py createsuperuser
    ```

6. Запуск:
    ```bash
    python manage.py runserver
    ```
